The participants' PC must have

    Ability to use video conferencing software like Zoom with headphones, a mic, and preferably a webcam.
    Ability to download from GitHub site.
    Some Office software.
    Ability to send and receive emails.
    A simple code editor like Visual Studio Code
    Java JDK 8 or later.

If you don't have any software from above on your PC, then  you can use the remote linux desktop in your browser.
	Go to https://cloudlabs.nuvepro.com/company/PragatiSoftware/home
	Ask trainer for login id and password, if haven't already received the same.
Refer the document HowToUseRemoteDesktop.pdf in this directory, for any issues.
